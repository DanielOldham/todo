About

To-do Web Application
This is a mini web application I built in one week as a knowledge assessment. After submitting, I have been adding extra features here and there for fun.
This web app is functioning as a stand-in for my portfolio website, which is currently under construction.
Feel free to take a look around or check out the Github!
Features
  • User accounts
  • User signup / login / logout functionality
  • To-do list page
  • To-do completion
  • Edit To-do functionality
  • Delete To-do functionality
  • List filter functionality (keyword, To-do status)
  • Responsive pagination and filtering with Server Sent Events
  • Unit tested views, forms, and models
Stack / Tools
  • Python Django 5
  • PostgreSQL
  • Bootstrap and Bootstrap Icons
  • Git / Github
  • Django Template Partials
  • Datastar
